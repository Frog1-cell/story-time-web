Установка Story Time SMP

Всё что нужно для игры на нашем сервере в одном установщике

Как устроена наша система установки

StoryTime-Manager

Наша консольная программа, которая автоматически устанавливает все необходимые моды и настройки

Автообновление

Менеджер сам проверяет обновления и предлагает установить их одним кликом

Безопасность

Все файлы проверяются на целостность перед установкой

Настройка под вас

Автоматическое определение папки Minecraft и версии игры

1 Скачайте установщик

Выберите версию для вашей операционной системы:

Windows

Portable .exe файл

Не требует установки. Просто скачайте и запустите

Linux

Исполняемый файл

Для Ubuntu, Debian, Fedora и других дистрибутивов

Arch Linux

AUR Package

Установка через AUR репозиторий

yay -S stm

2 Подготовьте Minecraft

Запустите ваш лаунчер

Поддерживаются: Official Launcher, TLauncher, SKLauncher, MultiMC и другие

Создайте новую сборку

Версия Minecraft: 1.20.1

Назовите её как угодно, например "Story Time SMP"

Запустите и закройте

Запустите созданную сборку один раз, дождитесь полной загрузки игры и закройте её

Важно: Убедитесь что у вас установлена Java 17 или новее

3 Установите моды через менеджер

1

Запустите StoryTime-Manager

Найдите скачанный файл и запустите его. Если нужно, дайте права администратора

2

Выберите опцию "Установить моды"

В открывшемся меню выберите первый вариант для установки модов

3

Выберите путь к Minecraft

Менеджер предложит найти папку автоматически или указать путь вручную

Рекомендуем выбрать автоматический поиск

4

Выберите тип сборки

Клиентская сборка

Рекомендуется! Оптимизирована для игры, содержит все необходимые моды

Серверная сборка

Только для создания своего клиента. Для игры на нашем сервере не требуется, только для интузиастов

5

Дождитесь завершения

Менеджер скачает все моды и зависимости. Процесс займет 1-5 минут в зависимости от скорости интернета

4 Проверьте установку и играйте

Проверьте папку с модами

Зайдите в .minecraft/mods/ и убедитесь что файлы загрузились

Запустите Minecraft

Запустите вашу сборку через лаунчер. В главном меню выберите "Мультиплеер"

Подключитесь к серверу

Добавьте сервер: story-time.playit.plus и присоединяйтесь к игре!

Частые проблемы и решения

Моды не скачались

Убедитесь что менеджер запущен с правами администратора. Проверьте подключение к интернету.

Игра не запускается

Проверьте версию Java (должна быть 17+). Убедитесь что в папке .minecraft достаточно места (минимум 2GB).

Менеджер не находит папку

Укажите путь вручную: скопируйте полный путь к папке .minecraft вашей сборки.

Обновление модов

Просто запустите StoryTime-Manager снова и выберите опцию "Обновить моды".
